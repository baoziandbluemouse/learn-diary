# 方法论

对数器（对拍）的实现
1. 你想要测的方法a（最优解）
2. 实现复杂度不好但容易实现的方法b（暴力解）
3. 实现一个随机样本产生器（长度随机，值随机）
4. 把方法a和方法b跑相同的输入样本，看看得到的结果是否一样
5. 如果有一个随机样本使得比对结果不一致，打印这个出错的样本进行人工干预，改对方法a和方法b
6. 当样本数量很多时对比测试依然正确，可以确定方法a（最优解）正确

关键是第5步，找到一个数据量小的错误样本，便于你去带入debug
然后把错误例子带入代码一步一步排查
Print，断点调试等
对数器可以用于验证贪心，找规律等
